# Proyecto — Módulo 3 JavaScript

## Descripción

Este repositorio contiene el desarrollo de las historias de usuario correspondientes al **Módulo 3 de JavaScript**, organizadas y trabajadas por semana.

Cada historia de usuario se desarrolla en una rama independiente y posteriormente se integra a la rama principal.

---

## Historias de Usuario por Semana

### Semana 1

- `Historia de Usuario 1`: Creación de un sistema interactivo de mensajes.

### Semana 2

- `Historia de Usuario 2` : Gestión de datos con objetos, Sets y Maps en JavaScript.

### Semana 3

- `Historia de Usuario 3` : Gestión dinámica del DOM con persistencia en el navegador.

---

## Flujo de trabajo

- La rama `main` contiene únicamente este archivo de documentación.
- Cada historia de usuario se desarrolla en su propia rama (`feature/*`).
- Las historias se integran a `main` mediante Pull Requests.
