# Sistema Interactivo de Mensajes – Semana 1

## Descripción del Proyecto
Este proyecto consiste en un sistema interactivo desarrollado en JavaScript que solicita al usuario su nombre y edad, valida la información ingresada y muestra mensajes personalizados según si el usuario es menor o mayor de edad.

El sistema se ejecuta en el navegador web utilizando ventanas emergentes (`prompt` y `alert`) y la consola del navegador para mostrar mensajes informativos y de error.

---

## Objetivo de la Actividad
El objetivo de esta actividad es reforzar los siguientes conceptos fundamentales de JavaScript:

- Declaración de variables usando `let` y `const`
- Manejo de tipos de datos
- Uso de `prompt()`, `alert()`, `console.log()` y `console.error()`
- Uso de estructuras condicionales (`if`, `else`)
- Validación de datos ingresados por el usuario
- Buenas prácticas de programación y comentarios en el código

## Descripción de Archivos

### `index.html`
Archivo HTML principal que permite ejecutar el sistema interactivo en el navegador.  
Incluye el archivo JavaScript mediante una etiqueta `<script>`.

---

### `sistema_interactivo.js`
Archivo JavaScript que contiene la lógica del sistema interactivo:

- Solicita el nombre del usuario
- Solicita y valida la edad ingresada
- Verifica que la edad sea un número válido
- Maneja errores de entrada utilizando `console.error()`
- Muestra mensajes personalizados según la edad
- Registra información validada usando `console.log()`

---

## Cómo Ejecutar el Proyecto
1. Abrir el archivo `index.html` en un navegador web.
2. Ingresar el nombre y la edad cuando el sistema lo solicite.
3. Los mensajes se mostrarán mediante ventanas emergentes (`alert()`).
4. Para ver mensajes de error o información adicional, abrir la **Consola del navegador** (`F12` o `Ctrl + Shift + I`).
